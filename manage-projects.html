Instructions

This page shows CRUD operations (Create, Read, Update, Delete) on project card data.

All of the CRUD operations use proper HTTP methods (GET, POST, PUT, DELETE) and log requests to your browser console. In local storage mode, changes will show up when you click the Load Local button on the projects page. Due to limitations with the remote server API (My JSON Server), changes made with the remote API are not able to persist between calls (but the requests are still performed as if they did).
